Course Description

This comprehensive online course introduces undergraduate students to the essential skills of data visualization, progressing from fundamental Excel techniques to advanced Python programming. Students will learn to transform raw data into compelling visual narratives that communicate insights effectively.

The course begins with Excel fundamentals, covering data management, formatting, and creating various chart types from basic bar charts to sophisticated heat maps and geographic visualizations. Students then transition to Python programming, learning core concepts before diving deep into powerful visualization libraries like Matplotlib and Seaborn.

By the course conclusion, students will master advanced tools including Pandas for data manipulation and Plotly for interactive visualizations. The hands-on approach ensures students gain practical experience with real-world datasets, developing both technical skills and design principles for effective data communication.

This course prepares students for careers in data science, business analytics, research, and any field where data-driven decision making is critical. The skills learned here form the foundation for advanced data science coursework and professional data analysis work.

Course Syllabus

1

Excel Fundamentals

Weeks 1-2
  • Data Management, Table Formatting, and Basic Excel Functions
  • Basic Excel Calculations and Working with Formulas
  • Introduction to Data Visualization and Excel Charting
  • Advanced Charts: Bubble, Box-and-Whisker, Waterfall, Funnel
  • Hierarchical Charts: Tree Maps, Sunburst, Radar Charts
  • Heat Maps and Geographic Visualizations
2

Python Programming Fundamentals

Weeks 3-5
  • Data Types: Numbers, Strings, Booleans, Lists, Dictionaries
  • Control Structures: Conditional Statements and Loops
  • Functions, Lambda Expressions, and Advanced Techniques
  • NumPy: Arrays, Indexing, and Mathematical Operations
  • Pandas: Series, DataFrames, Data Manipulation
  • Data Input/Output and Handling Missing Data
3

Python Data Visualization

Weeks 6-8
  • Matplotlib: Basic Plotting and Customization
  • Advanced Matplotlib: Subplots, Layouts, and Styling
  • Seaborn: Statistical Visualizations and Distribution Plots
  • Categorical Plots: Box, Violin, and Bar Plots
  • Matrix Plots: Heatmaps and Correlation Visualizations
  • Regression Plots and Advanced Seaborn Techniques
4

Advanced Visualization Techniques

Weeks 9-10
  • Built-in Pandas Visualization Capabilities
  • Plotly Express: Interactive Visualizations
  • SciPy: Statistical Analysis and Specialized Plots
